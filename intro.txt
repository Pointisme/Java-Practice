A. JAVA 특징

1. 운영체제에 독립적이다
  -자바가상머신을 통해서 가능한 일
  -JVM은 운영체제에 종속적,,,썬에서는 여러 운영체제에 설치할 수 있는 많은 버전의 JVM 제공

2. 객체지향언어
  -상속, 캡슐화, 다형성이 잘 적용된 순수한 객체지향언어

3. 비교적 배우기 쉬움
  -Small Talk이라는 객체지향언어에서 구문을, C++에서 연산자와 기본구문을 가져옴
  -간결하고 이해하기 쉬운 코드 작성
  -재사용성, 유지보수 용이

4. 자동 메모리 관리
  -자바를 이용하면 가비지컬렉터가 자동적으로 메모리를 관리해서 프로그래머가 따로 관리하지 않아도 된다.
  -자동으로 한다고 해서 무조건적으로 좋다는 것은 아님

5. 네트워크와 분산처리 지원
  -JAVA API 통해 네트워크 관련 프로그램을 쉽게 개발

6. 멀티쓰레드 지원
  -자바에서의 멀티쓰레드 프로그램은 시스템과 관계없이 구현 가능
  -JAVA API 사용

7. 동적 로딩 지원
  -프로그램 실행 시에 모든 클래스가 로딩되지 않고 필요한 시점에 클래스를 로딩하여 사용할 수 있다는 장점
  -일부 클래스가 변경되어도 전체 애플리케이션을 다시 컴파일하지 않아도 됨
