- public class의 이름과 소스파일명은 대소문자까지 모두 같아야된다.
- public class 존재 안하면 어떤 Class와 이름이 같아야된다.
- 모든 코드는 Class 내에 존재해야 된다.
- 하나의 Java 애플리케이션에는 main 메소드를 포함한 클래스가 반드시 하나는 있어야 한다.
- 변수를 선언한 이후부터는 변수를 사용할 수 있는데, 그 전에 반드시 초기화를 해줘야된다. (쓰레기값이 존재할 수 있어서)
- 클래스 이름의 첫 글자는 항상 대문자로 한다.
- 여러 단어로 이루어진 이름은 단어의 첫 글자를 대문자로 한다(lastIndexOf 같은 것)
- 상수의 이름은 모두 대문자로 한다. 여러 단어로 이루어진 경우 '_'로 구분한다.
